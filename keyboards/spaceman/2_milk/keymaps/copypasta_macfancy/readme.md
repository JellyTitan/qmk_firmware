# Tap Speed
Uses Tapping term for double and tripple tapping.
Tapping speed can be adjusted with TAPPING_TERM in config.h

## Key One
- 1 Tap: Copy
- 2 Taps: Cut
- 3 Taps: Tab

## Key Two
- 1 Tap: Paste
- 2 Taps: Paste as Value
- 3 Taps: Tab between programs
